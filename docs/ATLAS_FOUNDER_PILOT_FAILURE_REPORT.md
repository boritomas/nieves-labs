Date: 2026-07-13

This report records non-sensitive lessons from the founder-guided $50,000 SBA/CDFI funding pilot. It exists so Atlas can compress future campaigns into a five-step workflow instead of repeating account, portal, and field-entry loops.

Do not add SSN/ITIN values, dates of birth, passwords, MFA codes, private lender application IDs, bank account values, tax IDs, bankruptcy case numbers, identity documents, or private screenshots to this report.

## Five-step fix

1. Founder profile once: collect reusable company/founder/lender answers, but never store private identity values in source.
2. Lender triage and failover: rank SBA Microloan, CDFI, and verified community lenders; stop after verified blockers.
3. Portal field mapping: autofill only safe fields and mark private/certification fields as founder-only.
4. Submission proof: record lender, program, amount, status, submitted documents, evidence, and follow-up date.
5. Follow-up operating loop: track lender response windows, document requests, reminders, and fallback decisions.

## Security rule

Atlas may prepare application materials and track status. Atlas must not sign, certify, bypass MFA/CAPTCHA, store private identity values in source, or submit without explicit founder approval.
